# Power Two

Problem: Determine if a number is a power of two.

## Requirements

- Is the input number an int?
  - Yes
- Can we assume the inputs are valid?
  - No
- Is the output a boolean?
  - Yes
- Can we assume this fits memory?
  - Yes

## Example Usage

- None -> TypeError
- 0 -> False
- 1 -> True
- 2 -> True
- 15 -> False
- 16 -> True
