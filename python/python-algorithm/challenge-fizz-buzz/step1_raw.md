# Fizz Buzz

Problem: Implement Fizz Buzz.

## Requirements

- What is fizz buzz?
  - Return the string representation of numbers from 1 to n
    - Multiples of 3 -> 'Fizz'
    - Multiples of 5 -> 'Buzz'
    - Multiples of 3 and 5 -> 'FizzBuzz'
- Can we assume the inputs are valid?
  - No
- Can we assume this fits memory?
  - Yes

## Example Usage

```txt
* None -> Exception
* < 1 -> Exception
* 15 ->
[
    '1',
    '2',
    'Fizz',
    '4',
    'Buzz',
    'Fizz',
    '7',
    '8',
    'Fizz',
    'Buzz',
    '11',
    'Fizz',
    '13',
    '14',
    'FizzBuzz'
]
```
