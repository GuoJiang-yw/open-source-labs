# Sub Two

Problem: Find the difference of two integers without using the + or - sign.

## Requirements

- Can we assume the inputs are valid?
  - No, check for None
- Can we assume this fits memory?
  - Yes

## Example Usage

<pre>
* None input -> TypeError
* 7, 5 -> 2
* -5, -7 -> 2
* -5, 7 -> -12
* 5, -7 -> 12
</pre>
